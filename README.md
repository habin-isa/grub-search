# Grub search

### Be burdened with choice

### Search for similar restaurants using FourSquare, displayed using D3

### To run the project

- \$ git clone https://github.com/habin-isa/grub-search.git
- \$ cd grub-search
- \$ yarn
- \$ yarn start
